📈 今日の日本株ノート 記事投稿ツール

AIが生成した .md ファイル内容を貼り付けて、GitHubに直接投稿します

⚙️ GitHub トークン設定

GitHub → Settings → Developer settings → Personal access tokens (Fine-grained) で発行。
権限：Contents: Read and write のみ必要。設定はこのブラウザにのみ保存されます。

✅ トークンが保存されています
📋 .md ファイル内容 フロントマター＋本文をそのまま貼り付け

カテゴリ別：日経ノート → YYYYMMDD-nikkei.md ／ 深掘り → YYYYMMDD-deep.md ／ 投資メモ → YYYYMMDD-memo.md ／ 週次振り返り → YYYYMMDD-weekly.md

📄 ファイル名

形式：YYYYMMDD-[nikkei|deep|memo|weekly].md　内容を貼り付けてから「自動入力」を押すと pubDate と category から自動生成されます。
